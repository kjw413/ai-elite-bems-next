# 품목 × 에너지 영향 분석 리포트

**분석 기간**: 2024-01-01 ~ 2026-12-31
**대상 테이블**: `production_daily` × `energy_daily`
**대상 에너지**: 전력 (`total_power_kwh`), 연료 (`fuel_nm3`), 용수 (`water_ton`)

## 1. 분석 방법

- **Pearson / Spearman 상관**: 품목별 일별 생산량 vs 일별 에너지 사용량
- **Mix-adjusted 잔차 상관**: `mix_prod_kg` 영향을 회귀로 제거한 후 남는 상관
  → '체적 효과'를 제거하고 품목 고유의 부하 패턴만 본다
- **Lag(t+1) 상관**: 당일 생산 → 다음날 에너지 (지연 부하)
- **Ridge 회귀 (category 단위)**: 일별 카테고리 합계 → 에너지. 계수 = kg 당 에너지 사용량
- **Lasso 회귀 (item 단위)**: 200+ 품목 중 sparse 하게 영향 큰 품목만 선택
- **Bootstrap 95% CI**: 200회 리샘플로 회귀 계수 신뢰구간

## 5. 활용 가이드

- **신규 품목 도입**: 동일 (factory, category1, category2) 그룹의 `intensity_per_kg`
  를 기본 추정치로 사용. 계획 생산량 × intensity = 추가 에너지 수요 예상치.
- **기존 품목 증감**: Lasso top 품목이라면 해당 계수를, 그 외엔 카테고리 계수를 사용.
- **광주 주의**: `energy_daily.mix_prod_kg` 에 외주/임가공이 포함됨. 분석은 해당 계수에
  외주 부하가 일부 흡수돼 있어, 자사 품목만의 순영향은 카테고리 회귀가 더 보수적.
- **AI 보고서**: `app/services/item_energy_impact_service.py` 의 `estimate_impact()`
  를 통해 What-if 질문에 답할 수 있도록 룩업 JSON 을 활용.
