# 제조 데이터 관리 PoC 웹 시스템 기획서 (v1.1, 코드 기준 개정)

이 문서는 초기 PoC 기획서를 현재 구현 코드 기준으로 다시 정리한 요약본입니다. 상세 기준은 `docs/MASTER_SPEC_KR.md`를 우선 참고합니다.

---

## 1. 프로젝트 목적

본 프로젝트는 엑셀 기반 에너지 데이터를 로컬 웹 UI에서 업로드, 저장, 조회, 비교분석하기 위한 FEMS PoC 시스템입니다.

주요 목적:

- 공장별 에너지 데이터 통합 관리
- 전사 및 공장별 비교
- 월간 전년 대비 분석
- 업로드 이력 및 변경 이력 관리
- AI 실적 보고서 생성

## 4. 현재 웹 구조

- 대시보드(홈)
- 생산실적 분석
- 에너지 모니터링 (전력 사용량 / 연료·용수 사용량 / 원단위)
- 에너지 절감관리
- AI 에너지 분석 (에너지 사용 예측 / 에너지 실적 보고서)
- 데이터 업로드 (관리자 전용)
- 변경 이력 (관리자 전용)

현재 플레이스홀더:

- 절감관리 2개 화면 (절감 계획 관리 / 절감 실적 현황)

## 6. 검증 및 저장 규칙

| 항목 | 현재 처리 |
| --- | --- |
| 빈 값 | `0`으로 변환 |
| 음수 | 허용 |
| 숫자가 아닌 값 | 업로드 실패 |
| 동일 `(factory, date)` | 기존 데이터 업데이트 |
| 변경된 숫자 컬럼 | 감사 이력 기록 |

---

## 7. 집계 규칙

- 일별 데이터는 원본 실적 사용
- 월별 사용량은 일별 합계
- 월 원단위는 아래 공식으로 재계산

```text
월 원단위 = Σ(사용량) / Σ(mix_prod_kg / 1000)
```
