1. 가중 평균 원단위 산출 (Weighted Intensity): 원단위 컬럼을 단순 평균(AVG)하지 않으며, 반드시 SUM(사용량) / (SUM(생산량) / 1000) 공식을 사용하여 가중 평균을 산출한다. 생산량 단위는 반드시 ton으로 변환하여 계산한다.

2. 시각적 신호 표준화 (Visual Semantics): 리포트 작성 시 긍정적 성과(효율 개선, 비용 절감)는 파란색 강조(<span style="color:blue; font-weight:bold;">)로, 부정적 성과(원단위 증가, 효율 하락)나 경고는 빨간색 강조(<span style="color:red; font-weight:bold;">)로 엄격히 구분하여 표기한다.

3. 데이터 무결성 및 출처 명시 (Data Integrity): 모든 분석 결과에는 대상 사업장(남양주1, 김해 등)과 기준 연월을 명확히 명시하며, 수치 데이터 누락 시 "데이터 부족" 메시지를 출력하여 자의적 해석을 방지한다.

4. 비즈니스 맥락 연결 (Contextual Analysis): 단순한 수치 증감을 나열하는 데 그치지 않고, 생산량 변동, 설비 효율(냉동기, 공기압축기 등), 계절적 요인을 에너지 사용 변화와 연결하여 경영진이 즉각 의사결정을 내릴 수 있는 인사이트를 제공한다.
