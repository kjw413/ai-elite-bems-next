# 재공품 상관분석 리포트

**작성일**: 2026-04-22  
**목적**: 완제품 생산량(`믹스생산량[kg]`)만으로 설명되지 않는 에너지 사용량 변동을 보완하기 위해, 재공품 실적을 추가 피처 후보로 선별

## 1. 결론 요약

- 재공품 피처 추가 효과가 가장 클 것으로 보이는 조합은 **광주-연료**, **광주-용수**, **김해-연료**입니다.
- 중간 수준의 개선 가능성이 있는 조합은 **남양주1-용수**, **논산-용수**, **남양주2-연료**, **남양주2-전력**입니다.
- **광주-전력**과 **남양주2-용수**는 재공품과의 상관이 약해, 재공품 추가만으로는 성능 개선 폭이 크지 않을 가능성이 높습니다.
- 제공된 재공품 파일에서 **남양주1/남양주2 시트가 원본과 정리본 모두 완전히 동일**했습니다. 두 공장이 실제로 다른 생산 흐름이라면, 재공품 데이터 소스부터 다시 확인하는 것이 좋습니다.

## 2. 분석 기준

- 분석 기간: **2023-01-01 이후** 공통 구간
- 실제 분석 일자: **2023-01-02 ~ 2026-04-21**
- 분석 행수: 공장별 **822 영업일**
- 사용 데이터:
  - 에너지/생산량: `E:\Sampled DB\RawDB_에너지.xlsx`
  - 재공품 정리본: `E:\Sampled DB\DB_재공품.xlsx`
  - 재공품 품명 매핑: `E:\Sampled DB\RawDB_재공품.xlsx`
  - 기존 모델 성능표: `app/predictive model/energy usage/performance_test_results_v5.xlsx`

### 선별 로직

1. 공장별/타깃별로 재공품 품번과 에너지 타깃을 날짜 기준으로 매칭
2. 각 품번에 대해 아래 3개 지표 계산
   - `pearson_corr`: 타깃과의 선형 상관
   - `spearman_corr`: 순위 상관
   - `mix_adjusted_corr`: `믹스생산량[kg]` 영향을 제거한 뒤 남는 상관
3. 추천 후보는 아래 조건을 만족하는 품번만 사용
   - 재공품 발생일수 `>= 40일`
   - 커버리지 `>= 5%`
4. `|mix_adjusted_corr|`가 높은 순으로 정렬하되, 후보끼리 상호상관 `>= 0.85`이면 중복 피처로 보고 제외

`mix_adjusted_corr`를 핵심 지표로 본 이유는, 이미 모델에 들어가는 완제품 생산량과 겹치는 설명력보다 **추가 설명력**이 있는 재공품을 찾기 위해서입니다.
