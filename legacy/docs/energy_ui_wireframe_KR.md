# BEMS UI 구조 설계서 (코드 기준)

이 문서는 현재 `app/main.py`에 연결된 화면 구조를 기준으로 정리한 UI 문서입니다.

---

## 1. 전역 레이아웃

전역 UI는 다음 요소로 구성됩니다.

- 라이트 모드 단일 테마
- 왼쪽 사이드바 네비게이션
- Streamlit 메인 탭

사이드바 하단에는 아래 정보가 표시됩니다.

- 권한 배지 (🔓 root / 👁️ viewer) — IP 기반 자동 판정
- 현재 시각
- 사용자명
- 전체 레코드 수

---

## 2. 메인 내비게이션 구조

```text
BEMS
├─ 대시보드(홈)
├─ 에너지 모니터링
│  ├─ 사용량 통합
│  └─ 원단위
├─ 생산실적 분석
├─ AI 에너지 분석
│  ├─ 에너지 사용 예측      (viewer: 예측 실행 / 예측 이력 / 관리자: + 모델 관리 탭)
│  └─ 에너지 실적 보고서
├─ 데이터 업로드        (관리자 전용)
└─ 변경 이력            (관리자 전용)
```

---

## 3. 대시보드 화면

현재 메인 대시보드는 다음 섹션으로 구성됩니다.

1. `7일간 생산량 · 원단위 추이`
2. `월간 실적 전년비 현황`
3. `유틸리티 원단위 현황`
4. `생산량 및 사용량 현황`

화면 특성:

- 사업장 선택 지원
- 원단위/사용량/생산량 기준 비교 지원
- 전년 실적, 전년 동월, 전년 동기 비율 강조

---

## 4. 에너지 모니터링 화면군

### 4.1 사용량 통합

- 에너지원 라디오 선택: 전력 / 연료 / 용수 / 폐수
- 일별·기간별 사용량 추이와 KPI 카드
- 전력 선택 시 전체/냉동/공압기/기타 전력 분해
- 용수·폐수 선택 시 공장별 폐수/용수 비율 섹션 제공
- 전년대비 월별 차트와 누계 테이블

### 4.2 원단위

- 전력, 연료, 용수 원단위 중 선택 (폐수 원단위 폐기)
- 일별/기간별/전년비 비교
- 원단위는 단순 평균이 아니라 **Σ사용량 ÷ Σ생산량**으로 재계산
- 폐수는 원단위 대신 **폐수/용수** 비(폐수량/용수량, 소수점 2자리)로 관리 — 사용량 통합 화면의 '공장별 폐수/용수' 섹션 참조
---

## 5. AI 및 관리 화면

### 5.1 에너지 실적 보고서

- 공장/연월 선택
- 기존 보고서 조회
- 보고서 생성 또는 재생성
- 생성 결과 저장 및 재조회

### 5.2 데이터 업로드

- 관리자 전용
- Excel 업로드
- 업로드 결과 메시지
- 오류 상세 표
- 업로드 이력 표

### 5.3 변경 이력

- 관리자 전용
- 공장/날짜 필터
- 변경 로그 표

---

## 6. 메뉴에서 제외된 구형 화면

`절감 계획 관리`, `절감 실적 현황`은 현재 사이드바 메뉴에서 제외되어 있습니다. `app/main.py`의 현재 내비게이션 기준 정식 운영 화면은 생산실적 분석, 에너지 모니터링, AI 에너지 분석, 관리자 화면입니다.

생산실적 분석과 에너지 사용 예측은 본 구현되어 있습니다. 에너지 사용 예측은 권한에 따라 노출되는 탭이 다릅니다(viewer: 예측 실행/예측 이력 2탭, 관리자: + 모델 관리 탭).

---

## 7. 설계 메모

- 현재 메인 UI는 예전 단일 `에너지 실적 탭` 구조가 아니라 탭/서브탭 기반 구조입니다.
- 관리자 탭은 로그인 여부에 따라 동적으로 표시됩니다.
- 메인 내비게이션에 연결되지 않은 구형 페이지가 일부 존재할 수 있으나, 이 문서는 `app/main.py` 연결 기준만 설명합니다.
